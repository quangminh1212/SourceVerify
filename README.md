# SourceVerify — AI-Generated Content Detector

Advanced AI detection tool that analyzes images and videos to determine if they were generated by artificial intelligence. Built with Next.js 16 + TypeScript + Tailwind CSS 4.

## Features

- **Multi-Signal Analysis** — 8 independent analysis dimensions
- **Image & Video** — Support for JPEG, PNG, WebP, GIF, MP4, WebM
- **Client-Side Processing** — All analysis runs locally in browser
- **Privacy-First** — No data uploaded to servers
- **Instant Results** — Analysis in under 2 seconds

## Analysis Dimensions

| Signal | Description | Weight |
|--------|-------------|--------|
| Metadata Analysis | EXIF, software signatures, file naming | ×2.5 |
| Noise Uniformity | Block-wise noise variance patterns | ×2.0 |
| Frequency Analysis | High/low frequency ratio (FFT-like) | ×2.0 |
| Edge Coherence | Sobel edge detection quality | ×1.5 |
| Texture Consistency | Region-based texture variance | ×1.5 |
| Color Distribution | Histogram entropy analysis | ×1.2 |
| Symmetry Patterns | Mirror symmetry detection | ×1.0 |
| Compression Analysis | Bits-per-pixel ratio | ×0.8 |

## Tech Stack

- **Framework**: Next.js 16 (App Router)
- **Language**: TypeScript 5
- **Styling**: Tailwind CSS 4
- **Deployment**: Vercel

## Getting Started

```bash
npm install
npm run dev
```

## Deploy to Vercel

```bash
npx vercel --prod
```

Or connect GitHub repo → Vercel Dashboard → Import → Deploy.
